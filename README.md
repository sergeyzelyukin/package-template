# package-template

## Structure
```
.
├── LICENSE
├── README.md
├── bin
│   └── auxiliary1.sh
├── mypackage1
│   ├── __init__.py
│   ├── cli.py
│   ├── data
│   │   └── file1.txt
│   └── mymodules1
│       ├── __init__.py
│       ├── mymodule1.py
│       └── mymodule2.py
├── setup.cfg
├── setup.py
└── tests
    ├── conftest.py
    ├── test_mymodule1.py
    └── test_mymodule2.py
```

## To install dependencies
```
$ pip install -r requirements.txt
```
